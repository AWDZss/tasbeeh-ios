# Tasbeeh iOS

Native iOS wrapper for the Tasbeeh web app using SwiftUI and WKWebView.

## App Store Checklist

- Bundle identifier: `com.ibzah.tasbeeh`
- Display name: `تسبيح`
- Version: `1.0`
- Build: `1`
- Deployment target: iOS 16.0
- Target device: iPhone
- Encryption export compliance: non-exempt encryption disabled
- Privacy manifest: no collected data and no required-reason API usage declared

Before uploading to App Store Connect, open the project in Xcode on macOS, select your Apple Developer Team under Signing & Capabilities, then Archive from a generic iOS device destination.
